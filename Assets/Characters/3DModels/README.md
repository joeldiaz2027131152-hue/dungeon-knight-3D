# 3D Character Models

This folder is separate from the existing 2D sprite character folders.
Use it to test and integrate 3D characters gradually without removing the current sprite setup.

## Player/DarkKnight

- `Original`: raw exported/generated models.
- `MixamoPrep`: OBJ/ZIP files prepared for Mixamo upload.
- `Rigged`: models downloaded from Mixamo or Blender after rigging.
- `Animations`: animation FBX files such as idle, walk, attack, block, and roll.
- `Materials`: materials and textures used by the 3D version.

Keep the 2D sprites in `Assets/Characters/Player/DarkKnight` until the 3D version is fully working.
